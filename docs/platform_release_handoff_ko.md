# 플랫폼 반영 메모

## 2026-07-13: 계정별 복원 DB 및 플토 주문 가져오기

- Windows의 `main_desktop.dart`가 로그인 전에 기본 DB를 열던 문제를 수정했습니다. 이제 로그인한 계정의 DB만 사용하므로, 클라우드 복원 데이터가 올바르게 표시됩니다.
- 플토 주문보기에서 조회한 주문을 일반 주문보기로 가져오는 기능을 추가했습니다.
  - 이미 가져온 플토 주문은 주문번호 기준으로 중복 등록하지 않습니다.
  - 품목 연결이 있는 주문 상품은 주문 품목으로 저장합니다.
  - 품목 연결이 없는 상품은 주문 메모에 남깁니다.

## macOS / iOS 배포 시 확인할 사항

1. 이 변경이 포함된 동일한 Git 커밋으로 macOS와 iOS 앱을 다시 빌드합니다.
2. 테스트 계정으로 아이폰에서 클라우드 백업을 만들고, macOS 또는 iOS에서 복원한 뒤 품목·견적 수가 일치하는지 확인합니다.
3. 플토 주문보기에서 새 주문을 조회한 뒤 `주문보기로 가져오기`를 실행합니다.
4. 일반 주문보기에 새 주문이 한 번만 표시되는지, 다시 가져올 때 중복되지 않는지 확인합니다.
5. 품목 연결이 없는 주문 상품은 주문 메모에 남는지 확인합니다.
